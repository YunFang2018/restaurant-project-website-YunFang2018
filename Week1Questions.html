CSS Question

In how many ways can a CSS be integrated as a web page?
CSS can be integrated in three ways:
  1. Inline: Style attribute can be used to have CSS applied HTML elements.
  2. Embedded: The Head element can have a Style element within which the code can be placed.
  3. Linked/ Imported: CSS can be placed in an external file and linked via link element.
